## What does it do?

A simple react starter project using parcel. 

Featuring
- ReactJS
- react-router
- Babel
- ESlint (Airbnb)
- SASS
- Parcel dev environment

## Requirements
- ParcelJS - https://parceljs.org/
- NodeJS 10+ - https://nodejs.org/

## Install

    $ yarn install

## Run
    $ yarn dev

## Build
    $ yarn build
