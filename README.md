## What does it do?

A simple react starter project using parcel.

Featuring

- ReactJS
- react-router
- Babel
- ESlint (Airbnb)
- SASS
- Parcel dev server & build system
- Cypress integration test

## Requirements

- NodeJS (https://nodejs.org/)

## Install

    $ npm install

## Run integration tests

    $ npm test:cypress

## Run dev environment

    $ npm run dev

## Build

    $ npm run build
