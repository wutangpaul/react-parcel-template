## What does it do?

A simple react starter project using parcel. 

Featuring
- ReactJS
- react-router
- Babel
- ESlint (Airbnb)
- SASS
- Parcel dev environment

## Requirements
- NodeJS (https://nodejs.org/)
- Yarn (https://yarnpkg.com/en/)
- Parcel (https://parceljs.org/)

## Install
    $ yarn install

## Run
    $ yarn dev

## Build
    $ yarn build
